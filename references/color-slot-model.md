# 配色：主题色插槽模型

> 实测多个社保政务号样本发现：此类账号通常**无统一品牌色**，编辑按篇挑第三方模板换色。
> 据此采用「1 个可换主题色 + 固定语义色」插槽模型：结构固定，主题色每篇换一个。

## 色角色（4 类）

| 角色 | 色值 | 是否可换 | 用途 |
|---|---|---|---|
| 主题色 brand | 占位 `#ff6d4d` | ✅ 可换 | 小节标题、强调、卡片左边框、Q 标记、IP 口吻字、链接等**结构色**（✓ 正确结论等正向强调改用固定蓝，见下） |
| 警示红 warn | `#ff0000` | ❌ 固定 | ❎错误认知、注意/请注意、法律警告、诈骗警示 |
| 正确蓝 correct | `#5f9cef` | ❌ 固定 | ✓ 正确结论 / 正向结论（可算、可认定、正确办理方式等） |
| 中性 neutral | 文字 `#1F2329` / 辅助 `#8A8F99` / 卡片底 `#F5F6F8` / 白 `#ffffff` | ❌ 固定 | 正文、注释、卡片底 |

> **固定双语义色**：警示红 `#ff0000`（红=警示）+ 正确蓝 `#5f9cef`（蓝=正确）。两者均固定，不参与主题色替换。其余结构色（标题条、子标题芯片、举个例子、Q、大数字、时间线、点位名等）跟随主题色。

## 同色不疲劳：按"形式"分级，而非只靠色相

主题色会同时承担「结构」与「强调」，若都用同一种满色块/满色字，页面会"全是红的、找不到重点"。解法是**同一色相给不同的形式权重**，让每种信号长得不一样：

| 信号 | 形式 | 读者读到的 |
|---|---|---|
| 大分节标题 | 实底主题色条白字（文字长度底，唯一满色块） | 路标：新一节开始 |
| 子标题芯片（举个例子/01/02） | 满宽浅灰底 + 浅红字 + 主题色左边框 | 标签：这是个分类 |
| 正文关键词 | 主题色加粗字（无底） | 强调：这句重要 |
| 热点问答问题（Q1/Q2/Q3） | 浅红底 + 深字 + 红左边框 | 读者提问，与警示同红族但靠"Q 前缀"区分 |
| 提醒卡（温馨提示/注意！/请注意/IP郑重提醒/温馨提醒） | 外层用 `<table>` 容器；灰底 `#F5F6F8` + 浅灰细线边框 `1px solid #E5E6EB` + 圆角 `8px`、**无亮色边**；标题浅红字 `#E0494D` + 可选 💡 图标 + 可加主题色下划线（与字异色、`padding-bottom` 留呼吸感） | 与子标题芯片**不同款**（后者满灰底+主题色边）；微信后台会过滤 `<div>` 的 `border`，故用 `<table>` 保框；正文深灰纯文本、• 圆点分点；红色只作行内锚点，不整段/整行同色 |
| 正确结论 | 正确蓝字 | 全页唯一异色正向信号 |

要点：结构（实色条）与强调（彩色字）形式不同 → 不撞；警示/提醒类走**提醒卡**（灰底细线圆角框、无亮色边，`<table>` 承载），红色仅作行内锚点（❎ 标签、红字关键词、标题下划线），**不整段/整行同色**；红面积从"满屏条+红字"降为"几个小标记 + 少量词"。

## 主题色选用建议

- **红系主题（橙红/玫红/粉红/橙）与固定警示红同处红族**：结构信号（分节条、芯片边、强调词）和警示信号（红块）色相接近，只能靠"形式分级"区分，页面天然偏红、易疲劳。用红系主题时**务必搭配上文形式分级**——大分节条用文字长度底、子标题芯片用中性灰底、警示靠形状+红边，压低红面积。
- **警示型文章优先选非红主题**：文章含多个"请注意 / ❎ / 不能算"等警示信号时，优先用非红主题（社保蓝 `#438EDB` / 医保蓝 `#289DDA` / 青绿），结构与警示靠**色相**自然分开（蓝=结构、红=警示），层级最清楚、最不疲劳。
- 红系主题因品牌必须用也可，但接受"偏红"观感，并靠形式分级兜底。

## 主题色候选库

编辑每篇从中选一个，勿随意自定义。全号长期可收敛到 2–3 个主用色以积累品牌识别。

| 色值 | 调性 | 适合类型 |
|---|---|---|
| `#ff6d4d` | 橙红 | 默认/通用（指南、答疑） |
| `#be5960` | 玫红 | 政策解读（沉稳） |
| `#ff8587` | 粉红 | 新政/温和话题 |
| `#fa6729` | 橙 | 互动答疑（活泼） |
| `#438EDB` | 社保蓝 | A2 新政解读默认（社保政务号品牌色，与正确蓝 `#5f9cef` 区分） |
| `#289DDA` | 医保蓝 | 医保类文章备选（国家医保局官方 CHS 字形色） |
| 无 | 黑白 | 通知公告（公文，不设主题色） |

## 换色方式

**秀米/135（推荐）**：导入模板后用编辑器「主题色」一键换色。

**公众号原生编辑器**：粘贴前在源码里**查找替换占位主题色**为本篇主题色——A2 默认占位 `#438EDB`、其余模板默认占位 `#ff6d4d`；**警示红 `#ff0000`、正确蓝 `#5f9cef` 勿替换**（两者固定语义色）。

**脚本**：`scripts/apply_theme_color.py --input <file> --color <hex>` 自动替换占位主题色，跳过语义色。

## 纪律

- ❌ 主题色随意自定义 → ✅ 只从候选库选，色域可控
- ❌ 警示红/正确蓝跟着主题色换 → ✅ 双语义色固定（红=警示、蓝=正确），不随主题漂移
- ❌ 一篇混用多个主题色 → ✅ 一篇一个主题色贯穿全文
