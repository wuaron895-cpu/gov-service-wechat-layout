# Changelog

## v2.1.0 — 2026-07-12
- **警示/提醒类改为「提醒卡」，与子标题芯片区分**：`温馨提示 / 注意！ / 请注意 / IP郑重提醒 / 温馨提醒` 等收束类提示不再套"满灰底+主题色边"的子标题芯片，改为**灰底 `#F5F6F8` + 浅灰细线边框 `1px solid #E5E6EB` + 圆角 `8px`、无亮色边**；标题浅红字 `#E0494D` + 可选 💡 图标 + 可加主题色下划线（`border-bottom` 与字异色、`padding-bottom` 留呼吸感），正文深灰纯文本、`•` 圆点分点。
- **误区框（✓/❎）改为纯文本行内标记**：说明文字深灰 `#1F2329`，仅 ❎ 红 `#ff0000` / ✓ 蓝 `#5f9cef` 标记用色，不再套灰底卡。
- **子标题下正文纯文本 + 颜色只作行内锚点**：`(一)(二)…`、绑定规则、特别提醒等二级标题芯片下方正文一律纯文本、不套盒子；红色/蓝色只落关键词、标记、标签、标题下划线，不整段/整行同色。
- 同步：A2/C/D/A 模板的"请注意/注意！/郑重提醒"示例改提醒卡、误区框改行内标记；component-library.md（提醒卡组件 + 子标题正文规则 + 颜色锚点规则）、color-slot-model.md（信号分级表"警示"→"提醒卡"、要点同步）、SKILL.md 配色说明同步。

## v1.6.0 — 2026-07-12
- 解决"同色疲劳 / 全是红找不到重点"：主题色按**形式分级**区分信号，而非只靠色相。
  - 大分节条：实底主题色白字 → 默认改**文字长度底**（红块缩小，不再满屏）。
  - 子标题芯片（举个例子/01/02/Q/点位名等）：主题色红字 → **浅灰底 + 深字 + 主题色左边框**，读作"标签条"而非散落红字。
  - 警示块：浅红底 + 红字 → 加 **红左边框 + ⚠ 图标**，靠卡片形状从主题红里跳出（红=警示语义不变）。
- 新增 color-slot-model.md「同色不疲劳：按形式分级」节；component-library.md「三级小标题底色」与宽度模式默认改为文字长度底；SKILL.md 配色说明同步。
- 9 套模板统一改造（B 通知公告为黑白公文、本无主题色，无改动）。

## v1.5.0 — 2026-07-12
