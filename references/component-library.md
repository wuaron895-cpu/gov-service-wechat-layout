# 组件库与字号规范

## 字号层级 typeScale（公众号阅读基准）

| 层级 | 字号 | 行高 | 字重 | 用途 |
|---|---|---|---|---|
| 主标题 H1 | 18px | 1.8 | 700 | 文章主标题 |
| 小节 H2 | 16px | 1.8 | 700 | 误区N / 分节 / 案例N / Q 标记 |
| 正文 body | 15px | 1.8 | 400 | 段落 |
| 强调 strong | 15px | 1.8 | 700 | 关键词、法条、比例、金额 |
| 注释 note | 13px | 1.6 | 400 | 署名、来源、图注 |

## 字体 fonts

- 正文/标题：系统无衬线（苹方 / 微软雅黑 / Noto Sans SC 兜底）
- 关键词：同族加粗，不换字体，保证统一

## 标志性组件（招牌钩子）

- **人格化 IP**：口吻统一「{{IP名}}为大家解读了…{{IP名}}收到粉丝留言…」，品牌拟人核心。全号统一一个 IP 名。
- **听全文引导**：文首「点击🎧，可以听全文哦」
- **误区框（A）**：`误区N` 主题色标题 + ❎错误句（警示红）+ ✓正确句（正确蓝 #5f9cef/加粗）；❎/✓ 在正文里以**行内标记**呈现（说明文字深灰 `#1F2329`，仅标记/标签用色），不套灰底卡。
- **提醒卡（温馨提示 / 注意！ / 请注意 / IP郑重提醒 / 温馨提醒）**：灰底 `#F5F6F8` + 浅灰细线边框 `1px solid #E5E6EB` + 圆角 `8px`、**无亮色边**；使用 `<table>` 承载外框（微信公众号后台对 `<div>` 的 `border` 会过滤，对 `<table>` 的边框接受度更高）；标题浅红字 `#E0494D`、可加 💡 等中性图标、可加主题色下划线（与字异色、`padding-bottom` 留呼吸感），正文深灰纯文本。与子标题芯片不同款。
- **条件分支块（C）**：按「日期/情形」H2 分节，下挂子分节 `(一)(二)`
- **【】方括号关键词**：材料/字段名【】包裹强调（区别于纯加粗）
- **编号多形态**：公文「一、二」/ 途径「01 02」/ 提示「1.2.3.」
- **公告格式（B）**：标题重复（主标题=公告标题）+ 落款（单位+日期）+「特此公告。」
- **链接引导**：「可点击下方链接查看！」主题色可点
- **案例框（D）**：案例N +「案件介绍」「案件结果」子模块 +「知识链接」办理指引
- **IP 郑重提醒（D）**：提醒卡（灰底细线圆角框、无亮色边）收束警示
- **法律后果三块（D）**：1.刑事责任 2.行政处罚 3.民事赔偿 + 法条引用《刑法》《社会保险法》《XX条例》
- **Q&A 大号 Q 标记（E）**：Q1/Q2… 大号 Q（主题色）+ 问题加粗 + 解答
- **「举个例子」案例框（A2）**：具体人物 A/B 示例，把抽象规则具象化
- **视频引导**：「看完下方这个视频就更了解啦」
- **结尾号召（E）**：「有意愿参加…赶紧加入吧」
- **二维码下载/查看引导**：中段「长按识别下方二维码…」
- **置顶公众号引导**：「建议您将'{{公众号名}}'公众号置顶，只需3步哦~」
- **咨询话术 / footer**：文末「温馨提示 + 尊敬的参保人……拨打{{城市}}{{热线}}热线」
- **大数字高亮（F/G）**：`font-size:28px;font-weight:700;color:主题色` 居中展示关键数据（办件量 / 覆盖率 / 金额）。
- **时间线（F/G）**：灰底卡片 + `border-left:3px solid 主题色`，按日期列节点。
- **路线块（H）**：实底主题色大标题（白字）+ 浅底「打卡指南」串联条。
- **点位子卡片（H）**：点位名主题色加粗 + 描述 + 灰字字段行（接待 / 开放 / 电话）。
- **三级小标题底色（按形式区分，避免同色疲劳）**：大分节标题=实底主题色条白字（默认**文字长度底**，见下）；子标题/标签(举个例子/01/02/Q/点位名/案件介绍等)=**浅灰芯片 `#F5F6F8` + 浅红字 `#E0494D` + 主题色左边框**；热点问答问题(Q1/Q2/Q3)=**浅红底 `#FDECEC` + 深字 + 红左边框 `#ff0000`**（与子标题/警示同属"提示框"家族，靠浅红底+红边区分）。均用 `background-color/padding/border-left`，兼容直接粘贴公众号后台。
- **提醒卡（与子标题芯片区分）**：`温馨提示 / 注意！ / 请注意 / IP郑重提醒 / 温馨提醒` 等收束类提示**不套子标题芯片**，改用提醒卡（外层用 `<table>` 容器，灰底 `#F5F6F8` + 浅灰细线边框 `1px solid #E5E6EB` + 圆角 `8px`、无亮色边；标题浅红字 `#E0494D` + 可选 💡 图标 + 可加主题色下划线 `border-bottom` 与字异色且 `padding-bottom` 留呼吸感；正文深灰纯文本、`•` 圆点分点）。视觉上是一张圆角卡片，与“满灰底+主题色边”的子标题芯片区分开。改用 `<table>` 是因为微信后台对 `<div>` 的 `border` 会过滤，导致圆角方框消失。
- **子标题下正文规则**：`(一)(二)…`、绑定规则、特别提醒等二级标题芯片**下方正文一律纯文本**，不套盒子、不加色边；用 ✓/❎/• 等行内标记区分项。
- **颜色只作行内锚点**：红色/蓝色只落在关键词、标记、标签、标题下划线上，**不整段/整行同色**；警示信号靠提醒卡形状 + 浅红标题 + 行内红字承载，不靠满块红字。

## 小标题底色 · 宽度模式（section-title background width）

大分节标题的实底主题色条支持两种宽度，编辑按需选择：

- **满宽（默认，手机页面宽度）**：标题 `<p>` 用块级实底条，背景铺满正文宽度。
  `font-size:17px;font-weight:700;color:#ffffff;background:#ff6d4d;padding:9px 14px;`
- **文字长度**：背景只包裹标题文字（卡片式），视觉刺激更低，适合高饱和主题色。
  `display:inline-block;font-size:17px;font-weight:700;color:#ffffff;background:#ff6d4d;padding:9px 14px;margin:0 0 10px;`
  （若 `display:inline-block` 在个别客户端不换行，改用 `width:fit-content` 替代；两者都让背景贴着文字宽度。）

**选择规则**：模板默认主题（橙红 `#ff6d4d` / 玫红 `#be5960` / 粉红 `#ff8587` / 橙 `#fa6729`，均高饱和）→ 模板已内置**文字长度底**降刺激；冷静色（蓝 `#5f9cef` 作主题时）可用满宽；黑白（B 通知公告，无主题色）不适用。

> 换主题色时两种模式都只替换 `#ff6d4d` 即可，警示红 `#ff0000`、正确蓝 `#5f9cef` 不动。

## 内容类型骨架（9 类）

### A. 误区澄清 / 政策解读
头图 → 听全文 → 主标题 → 引言（抛出误区设问）→
[误区框 ×N：误区N / ❎错误 / ✓正确 + 法条引用] →
文末咨询话术 + 二维码 + 署名

### A2. 新政解读
头图 → 听全文 → 引言（新政几点新变化）→
一、待遇生效条件（含「举个例子」人物 A/B 框）→
二、办理材料（编号列表）→「请注意」提醒卡（灰底细线圆角框、无亮色边） →
三、办理途径（01 线上 / 02 线下，带操作步骤）→
四、热点问答（Q1/Q2/Q3，嵌套）→
置顶引导 → 温馨提示 + 咨询话术

### B. 通知公告
头图 → 听全文 → 公告标题（重复主标题）→
正文（主管部门通知 / 生效时间 / 影响说明）→
替代方案（编号 一、二）→
落款（单位 + 日期）+「特此公告。」→
温馨提示（补充块）→ 链接引导 →
置顶引导 → 温馨提示 + 咨询话术

### C. 办事指南
头图 → 听全文 → 引言（{{IP名}} + 上期回顾）→
关键提示框（提醒卡：灰底细线圆角框、无亮色边） →
条件分支块 ×N（按日期/情形 H2，子分节 (一)(二)；情形复杂可多层嵌套 + 材料编号清单）→
[编号 ① ② ③ + 【】关键词] →「注意！」提醒卡（灰底细线圆角框、无亮色边） →
二维码下载引导 → 置顶引导 → 温馨提示 + 咨询话术

### D. 案例故事 / 以案为鉴
头图 → 听全文 → 引言（零容忍态度定调）→
[案例N：案件介绍 / 案件结果 / 知识链接] ×N →
{{IP名}}郑重提醒（提醒卡：灰底细线圆角框、无亮色边） →
法律后果（1 刑事 / 2 行政 / 3 民事 + 法条引用）→
申领提示（编号）→ 置顶引导 → 咨询话术

### E. 互动答疑
头图 → 听全文 → 引言（{{IP名}}收到粉丝留言）→
[Q1 问题 + 解答] ×N → 注意事项(编号) + 相关链接（高频问答常见尾部）→ 二维码 / 视频引导 →
结尾号召 → 置顶引导 → 温馨提示 + 咨询话术

### F. 活动宣传 / 工作动态
头图 → 听全文 → 主标题 → 引言（背景/目的）→
【举措现场】大数字高亮 + 图墙占位 →
【延伸举措】+ 时间线（日期节点）→ 下阶段展望 →
置顶引导 → 温馨提示 + 咨询话术

### G. 年度盘点 / 数据战报
头图 → 听全文 → 主标题（数说XX这一年）→
大数字 hero ×3-4 → 成果分块（按业务线小标题 + 要点）→
大事记时间线 → 展望 →
置顶引导 → 温馨提示 + 咨询话术

### H. 路线攻略 / 分类合集
头图 → 听全文 → 主标题（钩子）→ 引言 →
路线① 大标题(实底主题色) + 打卡指南(浅底串联) + 点位子卡片 ×N(标题/描述/字段行) + 配套推荐 →
路线② … 路线N → 结尾号召 →
置顶引导 → 温馨提示 + 咨询话术
