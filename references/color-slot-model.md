# 配色：主题色插槽模型

> 实测多个社保政务号样本发现：此类账号通常**无统一品牌色**，编辑按篇挑第三方模板换色。
> 据此采用「1 个可换主题色 + 固定语义色」插槽模型：结构固定，主题色每篇换一个。

## 色角色（4 类）

| 角色 | 色值 | 是否可换 | 用途 |
|---|---|---|---|
| 主题色 brand | 占位 `#ff6d4d` | ✅ 可换 | 小节标题、强调、卡片左边框、Q 标记、IP 口吻字、链接 |
| 警示红 warn | `#ff0000` | ❌ 固定 | ❎错误认知、注意/请注意、法律警告、诈骗警示 |
| 正确绿 correct | `#2E9E5B` | ❌ 固定 | ✓ 正确结论 |
| 中性 neutral | 文字 `#1F2329` / 辅助 `#8A8F99` / 卡片底 `#F5F6F8` / 白 `#ffffff` | ❌ 固定 | 正文、注释、卡片底 |

## 主题色候选库

编辑每篇从中选一个，勿随意自定义。全号长期可收敛到 2–3 个主用色以积累品牌识别。

| 色值 | 调性 | 适合类型 |
|---|---|---|
| `#ff6d4d` | 橙红 | 默认/通用（指南、答疑） |
| `#be5960` | 玫红 | 政策解读（沉稳） |
| `#ff8587` | 粉红 | 新政/温和话题 |
| `#fa6729` | 橙 | 互动答疑（活泼） |
| `#5f9cef` | 蓝 | 案例警示（冷静理性） |
| 无 | 黑白 | 通知公告（公文，不设主题色） |

## 换色方式

**秀米/135（推荐）**：导入模板后用编辑器「主题色」一键换色。

**公众号原生编辑器**：粘贴前在源码里**查找替换**占位 `#ff6d4d` 为本篇主题色；警示红 `#ff0000`、正确绿 `#2E9E5B` **勿替换**。

**脚本**：`scripts/apply_theme_color.py --input <file> --color <hex>` 自动替换占位主题色，跳过语义色。

## 纪律

- ❌ 主题色随意自定义 → ✅ 只从候选库选，色域可控
- ❌ 警示红/正确绿跟着主题色换 → ✅ 语义色固定，红=警示 绿=正确，不随主题漂移
- ❌ 一篇混用多个主题色 → ✅ 一篇一个主题色贯穿全文
